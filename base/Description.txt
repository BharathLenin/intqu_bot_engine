Add src = The src attribute specifies the location (URL) of the external resource.
Add height = The height attribute specifies the height of the element, in pixels.
Add alt = The required alt attribute specifies an alternate text for an image, if the image cannot be displayed. The alt attribute provides alternative information for an image if a user for some reason cannot view it (because of slow connection, an error in the src attribute, or if the user uses a screen reader).
